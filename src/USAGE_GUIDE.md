### Adding a Chore

1. Tap the **Add** button (the blue circle in the center of the bottom nav)
2. Select **Chore** from the three options
3. Fill in the details:
   - **Task Name**: e.g., "Clean Kitchen"
   - **Assigned To**: Select the roommate responsible
   - **Duration**: Slide to estimate how long it will take (5-120 minutes)
   - **Due Date**: Pick a date from the calendar
   - **Due Time**: Select a specific time using the native time picker
4. Tap **Add Chore** to save
5. You'll see a success notification

**Pro Tip**: The app will highlight overdue tasks in red on the home screen!

### Logging a Meal

1. Tap the **Add** button
2. Select **Meal**
3. Enter details:
   - **Meal Name**: e.g., "Spaghetti Bolognese"
   - **Cooked By**: Who prepared the meal
   - **Who Ate?**: Check all roommates who enjoyed the meal
4. Tap **Add Meal** to log it
5. This contributes to the cook's fairness score!

### Recording a Purchase

1. Tap the **Add** button
2. Select **Purchase**
3. Fill in the information:
   - **Item Name**: e.g., "Groceries"
   - **Amount**: Enter the total cost
   - **Paid By**: Who made the purchase
   
4. **Choose Splitting Method**:
   
   **Option 1: Split Equally** (Most Common)
   - Divides the total evenly among selected roommates
   - Shows live preview of each person's share
   
   **Option 2: Split by Percentage**
   - Assign different percentages to each roommate
   - Useful when consumption varies (e.g., one person uses more)
   
   **Option 3: Custom Amounts**
   - Enter specific amounts for each person
   - For precise splitting scenarios

5. **Select Participants**:
   - Check each roommate who should share the cost
   - See the calculated split in the preview box

6. Tap **Add Purchase** to save

**Example**: 
- Groceries cost $45
- Split among all 3 roommates
- Each owes: $15.00

### Understanding Fairness Scores

**What are Fairness Scores?**
Fairness scores show how much each roommate contributes to household responsibilities, measured from 0-100.

**How They're Calculated:**
- Completed chores (weighted by effort)
- Meals cooked
- Shared expenses paid

**Color Coding:**
- ğŸŸ¢ **Green (65-100)**: Great contribution
- ğŸŸ  **Orange (55-64)**: Fair contribution

2. **Detailed View (Fairness Tab)**:
   - Sorted from lowest to highest
   - Visual progress bars with color coding
   - Bar chart for easy comparison
   - Lowest scorer highlighted with "Needs more tasks" label

**Pro Tip**: The person with the lowest score should consider taking on the next task!

## ğŸ› Troubleshooting

### Common Issues

**Q: A task shows as overdue but I completed it**
A: Make sure you tapped the checkbox to mark it complete. The system doesn't automatically detect completion.

**Q: The fairness score seems wrong**
A: Scores are calculated from logged activities only. Make sure all tasks, meals, and purchases are recorded.

**Q: How do I delete an activity?**
A: Currently, activities can only be marked complete, not deleted. This maintains an accurate history.

**Q: Can I edit an activity after adding it?**
A: The current version doesn't support editing. Double-check details before saving.

**Q: The splitting preview doesn't show my custom amounts**
A: Currently, live preview only works for "Split Equally" method. Percentage and Custom modes are coming soon.

## ğŸ“ Advanced Tips

### Optimizing Task Distribution

- **High-effort tasks** (45+ min): Rotate among all roommates
- **Low-effort tasks** (5-15 min): Can be assigned to lowest scorer
- **Urgent tasks**: Assign to whoever's available, regardless of score

### Meal Planning

- **Coordinate schedules**: Log who's eating to plan portions
- **Credit partial cooking**: If multiple people help, primary chef gets credit
- **Leftovers count**: If you reheated and served, log it

### Expense Management

- **Shared vs. Personal**: Only log truly shared expenses
- **Recurring bills**: Log monthly (Internet, utilities)
- **Bulk purchases**: Use "Split Equally" for household items
- **Unequal usage**: Use "Split by Percentage" for items like milk, if one person uses 60%
